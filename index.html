POST Create new event tag

https://nvn1.000webhostapp.com/api/event_tags
Body parameters
Name Required Value Description
api_token Yes String It's the token that you get after login or register successfully
name Yes String tag name

Example response when unauthenticated

Status: 401

{
    "message": "Unauthenticated."
}

Example response when the validation fail

Status: 422

{
    "message": "The given data was invalid.",
    "errors": {
        "name": [
            "The name field must be lowercase."
        ]
    }
}

Example response when you successfully create event page

Status: 201

{
    "message": "Event tag added."
}

POST Create new event page

https://nvn1.000webhostapp.com/api/events
Body parameters
Name Required Value Description
api_token Yes String It's the token that you get after login or register successfully
thumbnail Yes image file
tag Yes Number Id of event tag
content Yes String Content of event page
datetime Yes String Time of the event

{
    "message": "Unauthenticated."

            "The tag field is required."
        ],
        "content": [
            "The content field is required."
        ],
        "datetime": [
            "The datetime field is required."
        ],
        "thumbnail": [
            "The thumbnail field is required."
        ]
    }
}

Example response when you successfully create event page

Status: 201

{
    "message": "Event added."
}

GET [*] Delete event page

https://nvn1.000webhostapp.com/api/events/{id}/delete
Path parameters
Name Required Value Description
id Yes Number Id of event page
Query parameters
Name Required Value Description
api_token Yes String It's the token that you get after login or register successfully

Status: 401

{
    "message": "Unauthenticated."
}

Example response when specified page is not found

Status: 404

{
    "message": "Not found"
}

Status: 403

{
    "message": "You can't delete last event page."
}

}

Post Toggle page voting

https://nvn1.000webhostapp.com/api/votes/toggle

If you voted this page, this action perform unvote. If you haven't voted this page, this action perform vote.

Body parameters
Name Required Value Description
api_token Yes String It's the token that you get after login or register successfully
event_id Yes Number Event page id

Example response when Unauthenticated

Status: 401

{
    "message": "Unauthenticated."
}

Example response when the validation fail

Status: 422

{
    "message": "The given data was invalid.",
    "errors": {
        "event_id": [
            "The event id field is required."
        ]
    }
}

Example response when success vote page

Status: 201

{
    "message": "Voted."
}

Example response when success unvote page

Status: 201

{
    "message": "Vote deleted."
}
