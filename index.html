POST Create new event page

https://nvn1.000webhostapp.com/api/events
Body parameters
Name Required Value Description
api_token Yes String It's the token that you get after login or register successfully
thumbnail Yes image file
tag Yes Number Id of event tag
content Yes String Content of event page
datetime Yes String Time of the event

Example response when unauthenticated

Status: 401

{
    "message": "Unauthenticated."
}

Example response when the validation fail

Status: 422

{
    "message": "The given data was invalid.",
    "errors": {
        "tag": [
            "The tag field is required."
        ],
        "content": [
            "The content field is required."
        ],
        "datetime": [
            "The datetime field is required."
        ],
        "thumbnail": [
            "The thumbnail field is required."
        ]
    }
}

Example response when you successfully create event page

Status: 201

{
    "message": "Event added."
}

POST Register a new account

https://nvn1.000webhostapp.com/api/register

When you successfully register, a notification email will be sent to the email you used to register

Body parameters
Name Required Value Description
email Yes String
username Yes String
password Yes String
