POST Create new article

https://nvn1.000webhostapp.com/api/articles
Body parameters
Name Required Value Description
api_token Yes String It's the token that you get after login or register successfully
thumbnail No image file
name Yes String Article name
content Yes String Content of article

Example response when unauthenticated

Status: 401

{
    "message": "Unauthenticated."
}

Example response when the validation fail

Status: 422

{
    "message": "The given data was invalid.",
    "errors": {
        "name": [
            "The tag field is required."
        ],
        "content": [
            "The content field is required."
        ],
        "thumbnail": [
            "The thumbnail must be an image."
        ]
    }
}

Example response when you successfully create article

Status: 201

{
    "message": "New article added."
}
